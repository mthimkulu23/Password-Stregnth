The Most Common Poor Password Practices

Create Easy Passwords

Using predictable or simple passwords weakens your security.

Reuse Passwords

Using the same password across accounts increases breach risks.

Password Sharing

Never share your passwords with anyone, as doing so can compromise your privacy, security, and personal information .

Sharing Passwords Insecurely

Sending passwords over chat or email exposes you to interception.

Writing Down Passwords

Storing passwords in notebooks or spreadsheets puts your data at risk.

Skipping MFA

Not using multi-factor authentication is a missed layer of defense.

Using Short Passwords

Short passwords are easy to crack with brute force attacks.

Letting Browsers Save Passwords

Relying on browser password storage can be risky if the browser is compromised.
